This is a simple docker image for ComfyUI application.

## Usage

**Clone this repository**

```bash
git clone https://github.com/XavierBeheydt/docker-comfyui
cd docker-comfyui
```

**Run the services**

```bash
make up
```

**Install ComfyUI manager**

```bash
make install/manager restart
```
